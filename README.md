# BrickFramework

It's a simple 2D game engine. It's core is written in C++ by Gabriel Kaszewski and the editors are written in C# by Szymon Miś.

## Built With

* [SFML](https://www.sfml-dev.org/) - Simple and Fast Multimedia Library
